## tsconfig.json

```json
{
  "compilerOptions": {
    "allowJs": true,
    "checkJs": true,
    "noImplicitAny": true
  }
}
```

- noImplicitAny 옵션은 타입이 없더라도 any 타입이라도 넣어줘야한다는 옵션
